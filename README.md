# jittor-风之炼金术士华华-可微渲染新视角生成赛题

Jittor计图挑战热身赛代码仓库。队伍：风之炼金术士华华



## 简介

本项目为第二届计图挑战赛热身赛的代码实现，就是把basline跑了一次，不知道为啥没跑出来官方的成绩

## 安装

### 运行环境

#### 环境一

* winodws 10 专业版

* python >= 3.7

* jittor >= 1.3.0

* （可选）cuda >= 11.0

#### 环境二

* windows 10

* python >= 3.8

* jittor >= 1.3.0

* （可选）cuda >= 11.0
